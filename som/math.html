Procedure

  1. Initialize

  2. Grid to width * height * dimensions of input
  3. Starting neighbor width to max(gridWidth,gridHeight) / 2
  4. Starting learning rate to 0.05

  5. Weights random -1..1

  6. Prepare training data

  7. Scale(data) to mean centred on 0 and ideally range -1 to 1

  8. Loop per iteration

  9. Using inputs find the Best Matching Unit based on euclidean distance

\[ \sqrt{\sum_{i=1}^n(p_i-q_i)^2} \]

  1. Update weights in the region of the BMU (v) using scaling functions theta and alpha where theta is the nighborhood function and alpha is the per iteration learning rate.

\[ W_v(t+1) = W_v(t) + \theta(v,t) \times \alpha(t) \times (X(t) - W_v(t)) \]

  1. Use the Gaussian function as a radial basis function for neighbour weighting (x1,x2 are the row and column disctnaces from the BMU)

\[ f(x_1,x_2) = e^{-v} \]

\[ v = \sum_{i=1}^n\frac{x_i-c^2}{2w^2} \]
